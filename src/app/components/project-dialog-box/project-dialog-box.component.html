Features:

User Management:
Manages profiles for students, teachers, and alumni.
Forums:
A platform for interactions among students, teachers, and alumni, particularly focused on topics like placements, career guidance, and academic discussions.
Q&A Section:
Provides a dedicated space for students and teachers to ask and answer questions, helping resolve academic queries and foster collaboration.

Technologies Used:

Frontend:
The user interface is built using React.js, a part of the MERN stack, known for its flexibility and efficiency in building single-page applications.
Backend:
Node.js with Express.js handles the server-side operations, providing a fast and scalable backend.
Database:
MongoDB, a NoSQL database, is used for storing data. It allows for flexible schema design, making it well-suited for handling a variety of data types in a community system.
